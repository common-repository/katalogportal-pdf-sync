Automatically convert your uploaded pdf into media to flipbook and insert them as widget and shortcode

== Description ==

<h3>This plugin let you automatically synchronize your flipbooks with katalogportal.ch and use them as widget and shortcode inside your wordpress website</h3>
After uploading of the pdf inside 'media', your pdf are sent to katalogportal and your flipbook get prepared. The Path of the flipbook is inserted automatically to your pdf edit page.
For each flipbook you can set your own icon thumbnail before inserting the shortcode inside the editor and inside the widget area: Appearance->Widgets
The flip-book opens in an iframe over the page.

== Installation ==
You can manage all your menu pdf's / flipbook also directly inside the www.katalogportal.ch portal.
You need a 'Key' to be able to connect to Katalogportal, you can register in www.katalogportal.ch and get one.
In addition you need also your registered 'username' and your 'userid' from katalogportal.ch
Thereafter you can set this values under Settings->Katalogportal PDF Sync
The items are inserted inside your wordpress-page via webservice (http://www.katalogportal.ch/WebServiceKatalogPortal.asmx?WSDL)

= Using The WordPress Dashboard =

1. Navigate to the 'Add New' Plugin Dashboard
2. Select 'katalogportal-pdf-sync.zip' from your computer
3. Install
4. Activate the plugin on the WordPress Plugin Dashboard
5. Upload your PDF's to media
6. Place the desired 'Epaper Katalogportal' Widget into a Widget area through the 'Appearance -> Widgets' menu in WordPress.
7. Add your Epaper directly inside your editor using the Katalogportal icon on the top of your editor (a popup opens where you can select the desired uploaded catalog and your own thumbnail.

= Using FTP =

1. Extract 'katalogportal-pdf-sync.zip' to your computer
2. Upload the 'katalogportal-pdf-sync' plugin to your '/wp-content/plugins/' directory
3. Activate the plugin on the WordPress Plugins dashboard
4. Upload your PDF's to media
5. Place the desired 'Epaper Katalogportal' Widget into a Widget area through the 'Appearance -> Widgets' menu in WordPress.
6. Add your Epaper directly inside your editor using the Katalogportal icon on the top of your editor (a popup opens where you can select the desired uploaded catalog and your own thumbnail.
